# Memorijaa

This is a simple memory card-matching game developed as part of a faculty project. The game is designed to challenge the player's memory by matching pairs of cards. The game features a clean, custom design and smooth gameplay with basic functionality.

## Features

- **Card Matching:** Flip and match pairs of cards.
- **Custom Design:** Custom-designed cards and game interface.
- **Responsive Layout:** The game is responsive and works on both desktop and mobile devices.
- **Timer:** A basic timer to track how long it takes to finish the game.

## Technologies Used

- **HTML** for the structure of the game.
- **CSS** for styling and custom design.
- **JavaScript** for game logic (card flipping, matching, and timer).
